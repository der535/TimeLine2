
| Параметр          | Описание                                                                                                                                                                                                                                                                                    |
| ----------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Назначение        | Класс PlayerInvulnerable — статическая утилита для отслеживания состояния неуязвимости игрока в Unity. Определяет, находится ли игрок в режиме неуязвимости из‑за недавнего урона или рывка.                                                                                                |
| Принцип работы    | Содержит два флага: IsInvulnerableAfterDamage (неуязвимость после урона) и IsInvulnerableAfterDash (неуязвимость после рывка).<br>Метод IsInvulnerable() возвращает true, если активен хотя бы один флаг (через логическое ИЛИ), иначе — false.                                             |
| Как использовать? | Установите нужный флаг в true, чтобы включить неуязвимость: например, PlayerInvulnerable.IsInvulnerableAfterDamage = true.<br>Установите флаг в false, чтобы снять неуязвимость.<br>Проверьте общую неуязвимость вызовом PlayerInvulnerable.IsInvulnerable() — метод вернёт текущий статус. |
