
| Параметр          | Описание                                                                                                                                                                                                                                                                                                                                                                                                       |
| ----------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Назначение        | Класс ResurrectPlayer реализует воскрешение игрока в Unity (с ECS и Zenject): сбрасывает позицию, активирует игрока и включает управление.                                                                                                                                                                                                                                                                     |
| Принцип работы    | Через Zenject внедряются зависимости: GameEventBus, ActionMap и PlayerComponents.<br>В **Initialize()** подписывается на события RestartGameEvent и TurnToPlayModeEvent — оба вызывают метод Resurrect().<br>В Resurrect():<br>сбрасывает позицию игрока по X и Y (Z остаётся прежним);<br>активирует игрока через _playerComponents.SetActive(true);<br>включает управление через _actionMap.Player.Enable(). |
| Как использовать? | Установите нужный флаг в true, чтобы включить неуязвимость: например, PlayerInvulnerable.IsInvulnerableAfterDamage = true.<br>Установите флаг в false, чтобы снять неуязвимость.<br>Проверьте общую неуязвимость вызовом PlayerInvulnerable.IsInvulnerable() — метод вернёт текущий статус.                                                                                                                    |
