
| Параметр          | Описание                                                                                                                                                                                                                                  |
| ----------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Назначение        | Класс ResurrectPlayer реализует воскрешение игрока: сбрасывает позицию, активирует игрока и включает управление.                                                                                                                          |
| Принцип работы    | В **Initialize()** подписывается на события [[RestartGameEvent]] и [[TurnToPlayModeEvent]]— оба вызывают метод Resurrect().<br><br>В Resurrect():<br>- сбрасывает позицию игрока по X и Y<br>- активирует игрока<br>- включает управление |
| Как использовать? | Зарегестрировать в zenject                                                                                                                                                                                                                |
