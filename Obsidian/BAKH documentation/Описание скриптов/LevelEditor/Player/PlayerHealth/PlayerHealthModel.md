
| Параметр          | Описание                                                                                                                                                                                     |
| ----------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Назначение        | Класс PlayerHealthModel отвечает за хранение данных о здоровье игрока и реализацию базовой игровой логики, связанной с ним.                                                                  |
| Принцип работы    | Управляет состоянием здоровья — отслеживает текущий уровень относительно максимального, реагирует на события, влияющие на здоровье, и определяет момент гибели персонажа.                    |
| Как использовать? | Автоматически подключается и настраивается через систему зависимостей (Zenject) - вручную добавлять на объекты не требуется. Максимальное здоровье задаётся извне при инициализации системы. |
