
| Параметр          | Описание                                                                                                                                                                                                   |
| ----------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Назначение        | Класс PlayerHealthView отвечает за визуальное и звуковое представление состояния здоровья игрока.                                                                                                          |
| Принцип работы    | Отображает текущий уровень здоровья через UI‑элемент (слайдер), визуально отражает изменения (уменьшение/восстановление), сопровождает получение урона звуковым эффектом.                                  |
| Как использовать? | Разместите на том же GameObject, что и PlayerHealthPresenter, и заполните сериализуемые поля. Компонент автоматически синхронизируется с логикой игры через управляющий компонент (PlayerHealthPresenter). |
