
| Параметр          | Описание                                                                                                                                              |
| ----------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------- |
| Назначение        | Запускает и позиционирует систему частиц при рывке игрока.                                                                                            |
| Принцип работы    | В Play() устанавливает позицию частиц у игрока, запускает их (particle.Play()) и разворачивает по направлению движения (на основе PlayerMove).        |
| Как использовать? | Добавьте компонент, в Inspector укажите ParticleSystem. При вызове Play() частицы появятся у игрока, сориентируются по вектору движения и запустятся. |
