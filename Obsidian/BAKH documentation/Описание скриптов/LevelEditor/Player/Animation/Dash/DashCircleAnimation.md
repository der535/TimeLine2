
| Параметр          | Описание                                                                                                                                                                   |
| ----------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Назначение        | Анимирует визуальный круг рывка: плавно меняет радиус, толщину и цвет через DOTween.                                                                                       |
| Принцип работы    | При старте (Start) настраивает последовательность анимации (SetupSequence). При вызове Play() размещает круг у игрока (GetPosition) и запускает анимацию с нуля (Restart). |
| Как использовать? | Добавьте компонент на объект, настройте в Inspector dashCircle, dashCircleMaterial и параметры анимации (радиус, толщина, цвета). Вызовите Play(), чтобы запустить эффект. |
