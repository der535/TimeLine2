
| Параметр          | Описание                                                                                                                                                                  |
| ----------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Назначение        | Класс DashAnimation запускает и останавливает визуальные эффекты рывка                                                                                                    |
| Принцип работы    | Метод **Play()** активирует все эффекты: устанавливает масштаб рывка и запускает анимации круга, частиц и смены цвета.<br><br>Метод **Stop()** возвращает обычный масштаб |
| Как использовать? | Для запуска анимации рывка вызовите Play(), для остановки — Stop().                                                                                                       |
