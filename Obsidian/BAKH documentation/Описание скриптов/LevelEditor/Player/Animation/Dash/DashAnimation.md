
| Параметр          | Описание                                                                                                                                                                                                                                                                                                                                          |
| ----------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Назначение        | Класс DashAnimation запускает и останавливает визуальные эффекты рывка (dash) в Unity: управляет масштабированием, анимациями и цветом игрока.                                                                                                                                                                                                    |
| Принцип работы    | Содержит ссылки на компоненты эффектов рывка: dashCircleAnimation, _dashParticleAnimation, _dashPlayerColor и _playerMoveScale.<br>Метод **Play()** активирует все эффекты: устанавливает масштаб рывка и запускает анимации круга, частиц и смены цвета.<br>Метод **Stop()** возвращает обычный масштаб через _playerMoveScale.SetNormalScale(). |
| Как использовать? | Добавьте компонент DashAnimation на объект игрока.<br>В Inspector настройте ссылки на _dashCircleAnimation, _dashParticleAnimation, _playerMoveScale и _dashPlayerColor.<br>Для запуска анимации рывка вызовите Play(), для остановки — Stop().                                                                                                   |
