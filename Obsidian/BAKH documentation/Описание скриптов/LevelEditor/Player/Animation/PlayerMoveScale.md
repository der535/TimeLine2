
| Параметр          | Описание                                                                                                                                                                          |
| ----------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Назначение        | Класс PlayerMoveScale управляет масштабированием игрока меняет масштаб при движении/рывке и возвращает к базовому при покое.                                                      |
| Принцип работы    | Считывает вектор движения в update, если он не равен 0 то задаём скейл движения.<br><br>Ещё есть два метода SetDashScale и SetNormalScale вызываются из скрипта [[DashAnimation]] |
| Как использовать? | Работает автоматически, нужно лишь повесить скрипт на сцену                                                                                                                       |
