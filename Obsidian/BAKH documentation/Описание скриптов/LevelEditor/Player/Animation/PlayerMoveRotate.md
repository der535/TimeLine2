
| Параметр          | Описание                                                                                                   |
| ----------------- | ---------------------------------------------------------------------------------------------------------- |
| Назначение        | Класс PlayerMoveRotate отвечает за поворот игрока в направлении движения                                   |
| Принцип работы    | Считывает направление движения и посредство расчётов поворачивает игрока в зависимости вектора направления |
| Как использовать? | Работает автоматически, он должен просто висеть на сцене                                                   |
