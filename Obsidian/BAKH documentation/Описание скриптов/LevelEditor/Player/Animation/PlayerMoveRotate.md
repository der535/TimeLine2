
| Параметр          | Описание                                                                                                                                                                                                                                                                                                                                                                                                                             |
| ----------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| Назначение        | Класс PlayerMoveRotate отвечает за поворот игрока в Unity (с ECS и Zenject) в направлении движения на основе ввода с контроллера.                                                                                                                                                                                                                                                                                                    |
| Принцип работы    | Через Zenject внедряются зависимости: ActionMap (ввод) и PlayerComponents (данные игрока).<br>В **Update()** считывает вектор движения через _actionMap.Player.PlayerMove.ReadValue<`Vector2`>().<br>Если движение есть (sqrMagnitude > 0.01f), вычисляет угол поворота через Mathf.Atan2 (с коррекцией −90°).<br>Обновляет компонент LocalTransform сущности игрока: задаёт новую ориентацию по оси Z и записывает изменения в ECS. |
| Как использовать? | Добавьте компонент PlayerMoveRotate на объект игрока и убедитесь, что зависимости (ActionMap, PlayerComponents) зарегистрированы в Zenject. Поворот будет автоматически обновляться каждый кадр при наличии ввода движения.                                                                                                                                                                                                          |
