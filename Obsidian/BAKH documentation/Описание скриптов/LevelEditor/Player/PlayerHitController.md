
| Параметр          | Описание                                                                                                                                                                                                                          |
| ----------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Назначение        | Класс PlayerHitController обрабатывает получение урона игроком: активирует неуязвимость и запускает анимацию мигания на заданное время.                                                                                           |
| Принцип работы    | В методе **Initialize()** подписывается на событие PlayerTakeDamageEvent.<br><br>При получении события:<br>устанавливает флаг неуязвимости <br>запускает анимацию мигания <br>по завершении анимации сбрасывает флаг неуязвимости |
| Как использовать? | Зарегестрировать в zenject                                                                                                                                                                                                        |
