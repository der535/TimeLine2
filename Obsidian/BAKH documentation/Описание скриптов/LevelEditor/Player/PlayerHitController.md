
| Параметр          | Описание                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| ----------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Назначение        | Класс PlayerHitController обрабатывает получение урона игроком в Unity (с Zenject и EventBus): активирует неуязвимость и запускает анимацию мигания на заданное время.                                                                                                                                                                                                                                                                                               |
| Принцип работы    | Через Zenject внедряются зависимости: GameEventBus и PlayerHitAnimation.<br>В методе **Initialize()** подписывается на событие PlayerTakeDamageEvent.<br>При получении события:<br>устанавливает флаг неуязвимости PlayerInvulnerable.IsInvulnerableAfterDamage = true;<br>запускает анимацию мигания через _playerHitAnimation.Play() с длительностью TimeOfInvulnerability (2 с);<br>по завершении анимации (через колбэк) сбрасывает флаг неуязвимости. onFinish. |
| Как использовать? | Зарегистрируйте PlayerHitController в контейнере Zenject (Bind<`PlayerHitController`>().AsSingle()).<br>Вызовите метод Initialize() (например, в стартовой логике игры) — это активирует подписку на события урона.<br>Убедитесь, что PlayerHitAnimation корректно настроен и его метод Tick() вызывается каждый кадр.                                                                                                                                               |
