
| Параметр          | Описание                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    |
| ----------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Назначение        | Класс PlayerDeath отвечает за обработку смерти игрока в Unity (с использованием Zenject и EventBus): отключает управление, запускает эффекты и анимацию рестарта.                                                                                                                                                                                                                                                                                                                                                                                           |
| Принцип работы    | Через Zenject внедряются зависимости: GameEventBus, ActionMap, PlayerComponents, TimeLineRestartAnimation.<br>В **Start()** подписывается на события PlayerDeathEvent (вызывает Death()) и RestartGameEvent (сбрасывает флаг IsPlayerDeath).<br>Метод **Death()**:<br>устанавливает IsPlayerDeath = true;<br>отключает активность игрока через _playerComponents.SetActive(false);<br>деактивирует управление (_actionMap.Player.Disable());<br>запускает эффект смерти (deathParticles.Play());<br>запускает анимацию рестарта (_restartAnimation.Play()). |
| Как использовать? | Добавьте компонент PlayerDeath на объект игрока, настройте deathParticles в Inspector, убедитесь, что все зависимости зарегистрированы в Zenject. При отправке PlayerDeathEvent сработает логика смерти, при RestartGameEvent — флаг IsPlayerDeath сбросится.                                                                                                                                                                                                                                                                                               |
