
| Параметр          | Описание                                                                                                                                                                                                                                                                                                         |
| ----------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Назначение        | Класс PlayerDeath отвечает за обработку смерти игрока в: отключает управление, запускает эффекты и анимацию рестарта.                                                                                                                                                                                            |
| Принцип работы    | В **Start()** подписывается на события PlayerDeathEvent (вызывает Death()) и RestartGameEvent (сбрасывает флаг IsPlayerDeath).<br>Метод **Death()**:<br>устанавливает IsPlayerDeath = true;<br>отключает активность игрока<br>деактивирует управление<br>запускает эффект смерти <br>запускает анимацию рестарта |
| Как использовать? | Для работы скрипта он должен находится на сцене                                                                                                                                                                                                                                                                  |
