
| Параметр          | Описание                                                                                                                                  |
| ----------------- | ----------------------------------------------------------------------------------------------------------------------------------------- |
| Назначение        | Определяет сущность игрока в ECS: добавляет тег PlayerTag и компонент PostTransformMatrix при конвертации GameObject.                     |
| Принцип работы    | PlayerBaker конвертирует PlayerAuthoring в ECS‑сущность, присваивает ей PlayerTag и PostTransformMatrix с масштабом из Transform.         |
| Как использовать? | Добавьте компонент PlayerAuthoring на GameObject игрока — при сборке проекта он будет преобразован в ECS‑сущность с нужными компонентами. |
