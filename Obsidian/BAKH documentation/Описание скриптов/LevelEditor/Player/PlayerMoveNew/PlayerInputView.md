
| Параметр          | Описание                                                                                                                                 |
| ----------------- | ---------------------------------------------------------------------------------------------------------------------------------------- |
| Назначение        | Обрабатывает ввод игрока через Input System: передаёт события движения и нажатия пробела.                                                |
| Принцип работы    | При активации подписывается на ввод, при деактивации отписывается. Вызывает события: OnMovePerformed, OnSpacePerformed, OnSpaceCanceled. |
| Как использовать? | Для работы скрипта он должен находится на игроке                                                                                         |
