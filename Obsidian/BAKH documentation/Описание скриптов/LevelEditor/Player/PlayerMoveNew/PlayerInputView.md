
| Параметр          | Описание                                                                                                                                                        |
| ----------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Назначение        | Обрабатывает ввод игрока через Input System: передаёт события движения и нажатия пробела.                                                                       |
| Принцип работы    | При активации (OnEnable) подписывается на ввод, при деактивации (OnDisable) отписывается. Вызывает события: OnMovePerformed, OnSpacePerformed, OnSpaceCanceled. |
| Как использовать? | Добавьте компонент на объект ввода, подпишитесь на его события. При движении или нажатии пробела будут вызываться соответствующие обработчики.                  |
