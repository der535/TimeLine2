
| Параметр          | Описание                                                                                                                       |
| ----------------- | ------------------------------------------------------------------------------------------------------------------------------ |
| Назначение        | Синхронизирует позицию TrailRenderer (или другого визуального элемента) с позицией ECS‑сущности игрока.                        |
| Принцип работы    | В Update() получает позицию игрока из компонента LocalToWorld и присваивает её transform.position объекта с этим компонентом.  |
| Как использовать? | Добавьте PlayerTrailController на объект TrailRenderer. Он автоматически будет следовать за игроком, отображая его траекторию. |
