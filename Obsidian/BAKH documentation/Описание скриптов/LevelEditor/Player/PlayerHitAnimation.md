
| Параметр          | Описание                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    |
| ----------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Назначение        | Класс PlayerHitAnimation реализует анимацию мигания игрока при получении урона: меняет прозрачность материала по циклу в течение заданного времени.                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| Принцип работы    | При подписке на LevelLoadedEvent (в конструкторе) вызывает Initialize(), чтобы найти сущность игрока и получить его материал.<br><br>Метод **Play(float duration, Action onFinish)** запускает анимацию: задаёт длительность, сохраняет колбэк и активирует таймер.<br><br>**Tick()** вызывается каждый кадр — обновляет таймер, рассчитывает фазу мигания (прозрачный/непрозрачный) и меняет альфа‑канал цвета материала.<br><br>По завершении длительности вызывается **StopAnimation()**: сбрасывает флаг активности, восстанавливает полную непрозрачность и запускает колбэк onFinish. |
| Как использовать? | Зарегестрировать в zenject                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
