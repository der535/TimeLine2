
| Параметр          | Описание                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| ----------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Назначение        | Класс PlayerHitAnimation реализует анимацию мигания игрока при получении урона в Unity (с использованием ECS и Zenject): меняет прозрачность материала по циклу в течение заданного времени.                                                                                                                                                                                                                                                                                                                                                                                                               |
| Принцип работы    | При подписке на LevelLoadedEvent (в конструкторе) вызывает Initialize(), чтобы найти сущность игрока и получить его материал.<br>Метод **Play(float duration, Action onFinish)** запускает анимацию: задаёт длительность, сохраняет колбэк и активирует таймер.<br>**Tick()** вызывается каждый кадр — обновляет таймер, рассчитывает фазу мигания (прозрачный/непрозрачный) и меняет альфа‑канал цвета материала.<br>По завершении длительности (_timer >= _totalDuration) вызывается **StopAnimation()**: сбрасывает флаг активности, восстанавливает полную непрозрачность и запускает колбэк onFinish. |
| Как использовать? | Зарегистрируйте PlayerHitAnimation в контейнере Zenject.<br>Обеспечьте вызов метода Tick() каждый кадр (например, из Update() какого‑либо MonoBehaviour).<br>Для запуска анимации вызовите Play(duration, onFinish), указав длительность и действие по завершении.                                                                                                                                                                                                                                                                                                                                         |
