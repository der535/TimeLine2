

| Параметр          | Описание                                                                                                                                                                                                                                                                                                                                                                                |
| ----------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Назначение        | Класс PlayerHealth управляет здоровьем игрока: отслеживает урон, обновляет UI‑слайдер здоровья, запускает звук урона и обрабатывает смерть при нулевом здоровье.                                                                                                                                                                                                                        |
| Принцип работы    | В **Start()** инициализирует слайдер здоровья и устанавливает максимальное здоровье, подписывается на события<br>**TakeDamage()** уменьшает здоровье на 1, проигрывает звук, обновляет слайдер; если здоровье ≤ 0 — отправляет PlayerDeathEvent.<br>**RestoreHealth()** восстанавливает здоровье до максимума и обновляет слайдер (вызывается при рестарте и переходе в игровой режим). |
| Как использовать? | Для работы скрипта он должен находится на сцене                                                                                                                                                                                                                                                                                                                                         |
