
| Параметр          | Описание                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| ----------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Назначение        | Класс PlayerComponents управляет компонентами игрока в Unity (ECS + Zenject): инициализирует сущность, меняет коллайдер, активность, позицию и цвет материала.                                                                                                                                                                                                                                                                                                                                                                   |
| Принцип работы    | **Initialize()** — подписывается на LevelLoadedEvent, находит игрока по тегу PlayerTag, сохраняет материал в PlayerMaterial.<br>**ChangeCollider()** — создаёт и устанавливает новый BoxCollider.<br>**ChangeActive(bool)** — добавляет/удаляет тег PlayerTag и меняет прозрачность (альфа‑канал цвета).<br>**GetPosition()** — возвращает позицию игрока (float3) из LocalTransform.<br>**SetColor() / GetColor()** — устанавливают и возвращают цвет материала игрока.<br>Зависимости внедряются через Zenject (GameEventBus). |
| Как использовать? | Зарегистрируйте класс в Zenject (Bind<`PlayerComponents`>().AsSingle()), вызовите Initialize(), затем используйте методы для управления игроком.                                                                                                                                                                                                                                                                                                                                                                                 |
