
| Параметр          | Описание                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| ----------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Назначение        | Класс PlayerComponents управляет компонентами игрока в: инициализирует сущность, меняет коллайдер, активность, позицию и цвет материала.                                                                                                                                                                                                                                                                                                     |
| Принцип работы    | **Initialize()** — подписывается на LevelLoadedEvent, находит игрока по тегу PlayerTag, сохраняет материал в PlayerMaterial.<br>**ChangeCollider()** — создаёт и устанавливает новый BoxCollider.<br>**ChangeActive(bool)** — добавляет/удаляет тег PlayerTag и меняет прозрачность (альфа‑канал цвета).<br>**GetPosition()** — возвращает позицию игрока<br>**SetColor() / GetColor()** — устанавливают и возвращают цвет материала игрока. |
| Как использовать? | Для работы скрипта он должен находится на игроке                                                                                                                                                                                                                                                                                                                                                                                             |
