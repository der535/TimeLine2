| Параметр          | Описание                                                                                                                                                                |
| ----------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Назначение        | Этот скрипт отвечает за создание поля в инспекторе в которое можно спокойно переносить ассет сцены                                                                      |
| Принцип работы    | Этот скрипт использует кастомный сериализуемый класс для работы со сценами в Unity Inspector и позволяет выбирать сцены через ObjectField вместо строкового ввода       |
| Как использовать? | В любом скрипте где нужно передать сцену пишем - SceneField name;<br><br>Работатет при работе в редакторе самой игры, во время настройки кастомных уровней и их сценами |
