
| Параметр          | Описание                                                                                                                                       |
| ----------------- | ---------------------------------------------------------------------------------------------------------------------------------------------- |
| Назначение        | Этот скрипт отвечает за контроль шрифтов в игре.                                                                                               |
| Принцип работы    | Этот скрипт вешается на все объекты которые имеют компонент TextMeshProUGUI, в старте задаёт выбранный шрифт на основе FontNames и FontStorage |
| Как использовать? | Работает автоматически со старта, но он сам не объекты не добовляется, для это нужно использовать [[AddFontSetterTool]]                        |
